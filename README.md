# 🌸 Zinnia Magic – E-Commerce Website

A floral-themed **full-stack e-commerce platform** for selling Zinnia-inspired products such as home decor, apparel, jewelry, stationery, bath & body items, and accessories.

The platform allows customers to browse products, add items to a cart, and place orders, while administrators manage products, categories, employees, and inventory.

## ✨ Features

* User registration & login
* Browse products by category
* Search and filter products
* Add products to cart
* Checkout and order confirmation
* Admin dashboard for managing products and orders
* HR dashboard for managing employees
* Inventory dashboard for stock management

---

## ▶️ How to Run the Project

1. Install **XAMPP**
2. Start **Apache** and **MySQL**
3. Move the project folder to:

```
xampp/htdocs/zinnia
```

4. Import the database into **phpMyAdmin**
5. Open the project in your browser:

```
http://localhost/zinnia
```
